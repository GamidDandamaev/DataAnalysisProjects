# Realized Volatility Forecasting with Machine Learning

## Цель проекта

Проект посвящен прогнозированию реализованной волатильности акции Apple Inc.
(AAPL) на следующий торговый день по историческим OHLCV-данным.

Проект выполнен как учебный пет-проект бакалавра направления «Прикладная
математика и информатика».

## Данные

- Источник: Yahoo Finance
- Тикер: AAPL
- Период: 2017-01-01 — 2024-12-31
- Поля: Open, High, Low, Close, Volume

## Основная идея

В проекте прогнозируется не цена акции, а будущая реализованная волатильность.

В качестве оценки реализованной волатильности используется bipower variation
(BPV). Эта метрика менее чувствительна к отдельным резким скачкам цены, чем
обычная сумма квадратов доходностей.

Целевая переменная — реализованная волатильность на следующий торговый день.

## Признаки

Используются простые признаки, построенные только по прошлым данным:

| Признак | Описание |
|---|---|
| retLag1 — retLag5 | Лаги логарифмических доходностей |
| rvLag1 — rvLag5 | Лаги реализованной волатильности |
| rvWeekly | Средняя RV за неделю |
| rvMonthly | Средняя RV за месяц |
| highLowRange | Дневной диапазон High–Low |
| logVolume | Логарифм объема торгов |
| rollingVol | Скользящая волатильность |

## Модели

| Модель | Тип |
|---|---|
| Linear Regression | ML |
| Random Forest | ML |
| Gradient Boosting | ML |
| HAR | Эконометрическая модель |
| GARCH | Эконометрическая модель |

## Валидация и метрики

Используется walk-forward validation: модель обучается на прошлых данных и
прогнозирует следующий торговый день. Это позволяет избежать data leakage.

Метрики качества:

| Метрика | Описание |
|---|---|
| MAE | Средняя абсолютная ошибка |
| RMSE | Среднеквадратичная ошибка |
| QLIKE | Метрика для оценки прогнозов волатильности |

Чем ниже значение метрики, тем лучше модель.

## Результаты

| Модель | MAE | RMSE | QLIKE |
|---|---:|---:|---:|
| Linear Regression | 0.0700 | 0.1071 | 1.2179 |
| HAR | 0.0760 | 0.1252 | 1.2238 |
| Random Forest | 0.1190 | 0.3475 | 1.2588 |
| Gradient Boosting | 0.1167 | 0.3547 | 1.2652 |
| GARCH | 0.9032 | 1.0405 | 14.6407 |

Лучший результат показала Linear Regression. HAR-модель дала близкое качество,
что подтверждает важность лагов волатильности на разных горизонтах.

Random Forest и Gradient Boosting уступили простым моделям, возможно, из-за
переобучения или недостаточного объема данных. GARCH показал худший результат,
так как он моделирует условную волатильность доходностей, а не напрямую BPV.
